### **Snake Game Description**

The **Snake Game** is a Python-based arcade game developed using the **Turtle** graphics library. The player controls a snake using the arrow keys to collect food and increase the score. Each time the snake eats food, it grows longer. The game ends if the snake collides with the wall or its own body. This project demonstrates Python concepts such as **object-oriented programming (OOP), classes, loops, event handling, collision detection, and score management** while providing an interactive gaming experience.
